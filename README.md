# Universal Shift Register using QCA

## 📌 Description
This project presents the design and implementation of a 4-bit Universal Shift Register using Quantum Dot Cellular Automata (QCA) Designer.

## ⚙️ Tools Used
- QCA Designer

## 🔧 Modules Implemented
### Basic Gates
- AND, OR, NOT
- XOR, XNOR
- NAND, NOR

### Sequential Circuits
- D Flip-Flop
- D Latch

### Multiplexers
- 2:1 MUX
- 4:1 MUX
- 16:1 MUX

### Registers
- 4-bit Shift Register
- 4-bit Universal Shift Register

## 📊 Results
Simulation results are included in the Results folder.

## 🚀 Applications
- Nano-scale computing
- Low power digital systems
